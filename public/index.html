A C-like language for people who are tired of C/C++

PRSE has

  • syntax similar to C++, without all the fluff
  • standard commands for printing, numeric operations, and much more
  • dictionaries like Python
  • class declarations similar to C#

Frequently Asked Questions:

Q: How do I pronounce "PRSE"?

A: Personally, I pronounce it like "parse," since the program parses your code and outputs appropriately. Some prefer to pronounce it like "purse." You may also opt to spell it out, i.e. "Pee-arr-ess-ee."

Q: PRSE looks like an acronym. What does it stand for?

A: PRSE never really had any intentional acronym for it. The first one I came up with was Potentially Really Stupid Experiment. Other suggestions have been:

  • Parsed Readable Symbolic Expression
  • Programmable Response Significant Excellence
  • PRactical Systems languagE
There is no real definitive acronym, truth be told. Make one up yourself! Tweet it at me!
