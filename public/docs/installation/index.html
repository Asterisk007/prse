Building and Installing on Linux

Prerequisites/Dependencies:

PRSE requires that you have the following programs installed in order to build and run it:

  • bison: for building /src/prse.y, which contains the grammar rules of the language
  • flex: for building /src/prse.l, which lexes a given PRSE file
  • g++, clang, or any equivalent C++ compiler with support for the C++20 standard: for building the files in /src/lib/, and all other C++ files
  • make: for running the included Makefile in /src/

Building and installing the PRSE compiler

Open a terminal, and enter the following commands:
  1. git clone https://gitlab.com/Asterisk007/prse.git
    (alternatively, if you prefer to clone with SSH, use the command git clone git@gitlab.com:Asterisk007/prse.git)
  2. cd prse/src/
  3. make
  4. make install
    Note: you may need to run this using sudo
Finally, verify that the PRSE compiler is installed by running
  • prsec --version
You may need to re-log into your PC, or restart it.
