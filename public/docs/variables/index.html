Variables

Basic variable declaration:

Scalar data types:


                let name: type = value;
Where name is the name of the variable, type is its primitive (or user-created) type, and value is a value that matches the type. For instance, you might declare different variables with each available datatype as follows: 

                let foo: bool = true;
                let bar: int = 2;
                let baz: double = 6.28;
                let clive: char = 'e';
                let klaus: string = "a string";

Vector data types:


                let name: type[] = { value, value, value, ... };
The same as before applies here, except for two notable changes: first, the [] characters after type indicate that this is a vector; second, the { value, value, value, ... } syntax. Between the {} characters, you may type any valid value such as a function call, a variable, or a raw datatype, so long as the type matches that of the vector itself.
