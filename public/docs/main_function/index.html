The main function

As with all functions in PRSE, the main function is declared as follows: 


        func main()
There are a few rules associated with this function: first, main may only be a void return type, or an int return type. Second, the function definition cannot contain any parameters. If you wish to get arguments from the command line when you run your program, refer to getting arguments from the command line.
